# Rancid Tomatillos API

This is the back-end repo for the Rancid Tomatillos Front-End project. Documentation for the endpoints is found in the project spec.


## For Students:

There is no set up to worry about at this time. Requests can be made to `https://rancid-tomatillos.herokuapp.com/api/v2`. See the [project spec](https://frontend.turing.io/projects/module-3/rancid-tomatillos-v2.html#api-documentation) for more info.
